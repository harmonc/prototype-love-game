# Controls

## Movement
- **WASD / Arrow Keys** - Pan the view

## Gameplay
- **Space** - Step the game forward
- **Left Click** - Place object at cursor
- **Right Click** - Remove object at cursor
- **Mouse Wheel** - Zoom in/out

## Objects
- **T** - Toggle between circle and triangle
- **R** - Rotate triangle direction (up → right → down → left)
- **1-0** - Select inventory slot (0 = slot 10)

## UI
- **G** - Toggle debug info
- **H** - Toggle grid visibility
- **Escape / P** - Pause menu

## Debug Menu
- **F1** - Open/close debug menu
- **W/S or Up/Down** - Navigate menu
- **Enter / Space** - Select option
  - Clear Grid - Remove all objects
  - Clear Save & Reload - Delete save and start fresh
